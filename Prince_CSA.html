Java Programming Basics

Brief Description of Project

Course: AP Computer Science Applications (CSA)
Duration: August 29,2019 to September 7,2019
Objective: Create a Seating Chart in Greenfoot
Type of Project: Individual Project
Project Title: AP CSA Period 3 Seating Chart

This project was the first project of the year, which made us utilize programming basics, we gained throughout the summer by watching Youtube videos about Java programming (topics include: Java syntax, loops, arrays, object-oriented programming, inheritance, condition statements). The main objective of this project was to create a seating chart for our class. The client of the project would be our teacher as it served as a way to check where the student sat, how to pronounce the students' names, and interact with the student via a question and answer section. No written documentation was done for this project. Instead, a ll of the code that was written, had Java documentation, if other students wanted to understand our code. The rubric, used for this project mainly focused on individual aspects we needed to complete such as having our picture in the correct location that corresponded to where we sat, having an animation when our image was clicked, having a sound byte for the teacher to remember our name pronunciation, etc. One skill acquired from the project was reinforcing my knowledge about the basics of programming, as this was my first time programming in Java. Another strength, which I was able to garner was learning that it was essential to communicate with my peers. Even though the project was individual, we all had to push to one Git Repository and ensure that we were in the right location, before pushing our code. The Git Master (person in charge of merging all our code), coordinated everyone's code and was able to finalize the seating chart. Through this project, I was able to create a place in the seating chart, that corresponds to where I sat, that met all the requirements, and more as I also completed the extra credit. Above, is the video of the seating chart, demonstrating how it can be a helpful tool for the teacher.

Java Sound Library

Brief Description of Project

Course: AP Computer Science Applications (CSA)
Duration: January 7, 2020 to January 22, 2020
Objective: Analyze a Java Library
Type of Project: Partner Project
Project Title: Sound Library

This project was the first project of the second semester, where we had to research and learn about various Java libraries. The main objective of this project was to create a 7-10 presentation about a select Java library. The client of the project was our classmates as they were able to learn about Java libraries that they did not know about previously. Additionally, to explain the capabilities of the library, we had to use the library to create a software application. The documentation for this project was in the format of a task list, where we had to indicate what our duties were in the project. Also, we had to color-code our tasks to indicate, whether the sections were complete, incomplete, canceled or in progress. The rubric used for this project mainly focused on individual aspects such as our presentation style, the application we created and ensuring that our presentation was well-timed. One skill acquired from the project was learning about a new library, which I never knew about. Another skill that I was able to hone through this project was communicating with a diverse group of students and ensuring they learn about the library in depth. One major problem that we had to overcome was figuring out which application to create and how it would best demonstrate some of the library capabilities. We soon settled on creating an application surrounding music, given that we both have musical backgrounds. In our app, we catered it to musicians who would like to tune their instruments, catered many towards violinists and pianists. The app allows for musicians to play the note once, or play it multiple times to ensure that their instrument is tuned correctly to play their musical pieces accurately.
